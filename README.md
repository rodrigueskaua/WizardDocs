# WizardDocs

WizardDocs é um sistema de chat inteligente que permite interagir com arquivos PDF, combinando técnicas de **RAG (Retrieval-Augmented Generation)** e **ChromaDB** para busca semântica avançada.

## 🔹 Funcionalidades

- 📂 **Upload de PDFs** – Envie documentos para análise.  
- 💬 **Chat Interativo** – Faça perguntas sobre o conteúdo dos PDFs.  
- 🔍 **Busca Semântica** – Identificação de trechos relevantes através de embeddings.  
- 🤖 **Respostas Contextualizadas** – Geração de respostas inteligentes baseadas nos documentos.  
- 🚀 **Desempenho Otimizado** – Uso de **FastAPI** e **ChromaDB** para respostas rápidas.  
- 📡 **APIs e Integração** – Conexão com outras aplicações para automação e análise de dados.  

## 🛠 Tecnologias

- **Python** – Linguagem principal do projeto.  
- **FastAPI** – API de alto desempenho.  
- **ChromaDB** – Banco de dados vetorial para busca semântica.  
- **RAG (Retrieval-Augmented Generation)** – Técnica para geração de respostas.  
- **Docker** – Containerização para fácil implantação.  
- **Transformers & Sentence Transformers** – Modelos para embeddings e processamento de linguagem natural.
